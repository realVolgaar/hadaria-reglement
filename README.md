# Wiki Hadaria

Wiki officiel du serveur Hadaria, propulsé par [MkDocs Material](https://squidfunk.github.io/mkdocs-material/).

## Structure

```
docs/
├── index.md                  # Page d'accueil
├── reglement-general.md      # Règlement général
├── reglement-events-pvp.md   # Règlement Évents PvP
├── reglement-farm.md         # Règlement Farm
└── reglement-pillage.md      # Règlement Pillage
```

## Modifier le wiki

Éditez simplement les fichiers `.md` dans le dossier `docs/`. Le site se met à jour automatiquement à chaque push sur `main`.

## Prévisualiser en local

```bash
pip install mkdocs-material
mkdocs serve
```

Puis ouvrez [http://localhost:8000](http://localhost:8000).
